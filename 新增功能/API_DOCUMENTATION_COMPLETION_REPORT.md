## 🔧 工具集成

### Postman 导入

```bash
# 使用 openapi.yaml 生成 Postman Collection
openapi2postmanv2 -s docs/swagger/openapi.yaml -o postman_collection.json
```

然后在 Postman 中导入 `postman_collection.json`

### Swagger UI

```bash
# 安装 swagger-ui-express
npm install swagger-ui-express swagger-jsdoc

# 访问交互式文档
# http://localhost:3000/api-docs
```

### 客户端代码生成

```bash
# 使用 OpenAPI Generator
# 生成 Android (Kotlin)
openapi-generator-cli generate -i docs/swagger/openapi.yaml \
  -g kotlin -o ./kotlin-client

# 生成 JavaScript (Axios)
openapi-generator-cli generate -i docs/swagger/openapi.yaml \
  -g typescript-axios -o ./js-client

# 生成 Swift
openapi-generator-cli generate -i docs/swagger/openapi.yaml \
  -g swift5 -o ./swift-client
```
