# 标签管理 API 文档

## 📋 模块概述

标签管理模块提供内容标签的创建、管理、批量操作功能，用于内容分类和组织。

**接口数量**: 8个
**基础路径**: `/api/v1/tags`
**认证要求**: 所有接口均需要认证 ✅

---

## 📑 主要接口列表

### 标签CRUD（5个接口）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取所有标签 | GET | `/` | 获取所有标签列表 |
| 创建标签 | POST | `/` | 创建新标签（管理员） |
| 更新标签 | PUT | `/:id` | 更新标签（管理员） |
| 删除标签 | DELETE | `/:id` | 删除标签（管理员） |
| 获取内容标签 | GET | `/content/:id/tags` | 获取内容的标签 |

### 标签关联操作（3个接口）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 添加标签 | POST | `/content/tags/add` | 为内容添加标签 |
| 移除标签 | POST | `/content/tags/remove` | 从内容移除标签 |
| 批量操作 | POST | `/content/tags/batch` | 批量操作标签 |

## 核心接口说明

### 9.1 标签数据结构

```json
{
  "id": "uuid",
  "name": "美食",
  "color": "#1890ff",
  "description": "美食相关内容",
  "usage_count": 25
}
```

---

### 9.2 为内容添加标签

**端点**: `POST /api/v1/tags/content/tags/add`

**请求参数**:
```json
{
  "content_id": "uuid",
  "tag_ids": ["tag1", "tag2"]
}
```
